# API Routes

Τα routes της εφαρμογής προστίθενται εδώ στα επόμενα βήματα:

- `auth.js` — login/logout (ΒΗΜΑ 2)
- `agents.js` — CRUD agents + constraints (ΒΗΜΑ 2)
- `period.js` — περίοδος, άδειες, αιτήματα ρεπό (ΒΗΜΑ 3)
- `schedule.js` — generator, preview, αποθήκευση (ΒΗΜΑ 4-5)
- `export.js` — Excel export (ΒΗΜΑ 6)
